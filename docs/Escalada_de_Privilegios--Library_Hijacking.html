Library Hijacking


Library Hijacking es muy similar al PATH Hijacking solo que jugando con las librerias por ejemplo en este caso de Python

Cuando ejecutamos un script con Python, la primera ruta en la que busca las librerias es la actual por lo que podemos crear una libreria en nuestro directorio actual que ejecute una shell y al tener permiso el binario de python nos la ejecutara como root
