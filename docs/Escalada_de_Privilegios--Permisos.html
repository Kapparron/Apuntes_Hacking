Permisos


Para los permisos rwx rwx rwx → para calcularlos se contaria con 1 y 0

rwx rwx rwx r-x rwx --x
7 7 7 5 7 1
111 111 111 101 111 001

Cuando hay un uno significa que se eleva un 2 a la posicion en la que este es decir si por ejemplo es 111 seria 2^2+2^1+2^0 que seria igual a 7, otro ejemplo seria 101 que seria 2^2+0+2^0 que seria igual a 5

Podremos usar el comando FIND para buscar archivos con permisos de escritura y podemos usar el grep para fitrar los resultados:

┌──(kali@kali)-[~]
└─$ find \-writable 2> /dev/null


Ver la configuracion de sudoers sobre un usuario:

┌──(kali@kali)-[~]
└─$ sudo -l
