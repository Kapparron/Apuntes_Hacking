Enumeracion Web



LISTAR INFORMACIÓN RELEVANTE DE UN SERVIDOR WEB

┌──(kali@kali)-[~]
└─$ whatweb http://10.0.0.1


VIRTUAL HOST ROUTING

Te permite contrar con varios servidores web en una maquina, para dirigirnos a otro server lo indicamos en el /etc/hosts

127.0.0.1 localhost
10.0.0.1 web.com


ANALISIS DNS

┌──(kali@kali)-[~]
└─$ gobuster dns -d web.com -w diccionario.txt -i


WAFW00F

Sirve para identificar si el servidor web tiene un WAF (Web Aplication Firewall)

┌──(kali@kali)-[~]
└─$ wafw00f http://10.0.0.1


WPSCAN

┌──(kali@kali)-[~]
└─$ wpscan --url http://10.0.0.1 -e vp,u
