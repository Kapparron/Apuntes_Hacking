SUID


Los SUID son permisos que nos permiten ejecutar temporalmente como el usuario definido como propietario del archivo.

rwsr-x--x la “s” que aparece significa que tenemos permisos de SUID

ASIGNAR PERMISOS SUID

┌──(kali@kali)-[~]
└─$ chmod 4755 archivo


LISTAR ARCHIVOS CON PERMISOS SUID

┌──(kali@kali)-[~]
└─$ find \-perm -4000 2> /dev/null


┌──(kali@kali)-[~]
└─$ find / -user root -perm -4000 -exec ls -ldb {} \; 2> /dev/null


┌──(kali@kali)-[~]
└─$ find / -user root -perm -6000 -exec ls -ldb {} \; 2> /dev/null


RECORDAR!!!! → Cuando ejecutamos una bash con SUID es importante la flag “-p" para iniciar esa bash como root
