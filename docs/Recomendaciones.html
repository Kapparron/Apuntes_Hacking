SANITIZAR TTY

Desde la maquina hackeada escribimos lo siguiente

┌──(kali@kali)-[~]
└─$ script /dev/null -c bash


Dejamos el proceso en segundo plano con CTRL+Z y volveremos a nuestro host

┌──(kali@kali)-[~]
└─$ stty raw -echo

┌──(kali@kali)-[~]
└─$ fg

┌──(kali@kali)-[~]
└─$ reset


A continuacion nos pedira una terminal es recomendable xterm, ahora dentro de la maquina cambiaremos las siguientes variables de entorno

┌──(kali@kali)-[~]
└─$ export TERM=xterm

┌──(kali@kali)-[~]
└─$ export SHELL=bash


Ahora en nuestro host comprobamos las proporciones de nuestra consola con el siguiente comando

┌──(kali@kali)-[~]
└─$ stty -a


De vuelta a la maquina introducimos el resultado del comando anterior para ver todo en pantalla completa

┌──(kali@kali)-[~]
└─$ stty rows 52 columns 187
